# 📘 Assignment: Python Classes

## 🎯 Objective

Learn how to define and use classes in Python to model real-world objects and behaviors.

## 📝 Tasks

### 🛠️ Define a Simple Class

#### Description
Create a class named `Car` that represents a car with attributes for `make`, `model`, and `year`. Implement a method to display the car's information.

#### Requirements
Completed program should:

- Define a class `Car` with `make`, `model`, and `year` attributes
- Include a method `display_info()` that returns or prints the car's details in a readable format
- Create an instance of `Car` and call `display_info()` to show expected output

### 🛠️ Add Methods and Interactions

#### Description
Extend the `Car` class to track and update mileage, and add methods that interact with the class state.

#### Requirements
Completed program should:

- Add a `mileage` attribute to the `Car` class (default 0)
- Implement `update_mileage(new_mileage)` which updates mileage only if `new_mileage` is greater than current mileage
- Implement `display_mileage()` that returns or prints the current mileage
- Demonstrate creating a `Car` instance, updating its mileage, and displaying the updated mileage
